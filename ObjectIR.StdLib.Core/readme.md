# ObjectIR Standard Library - Core

Welcome to the official ObjectIR Standard Library - Core module!
This module provides a collection of fundamental interfaces and types that form the backbone of the ObjectIR ecosystem.
It includes essential components such as basic data types, utility functions, and core interfaces that are designed to be used across various ObjectIR Runtime implementations.
